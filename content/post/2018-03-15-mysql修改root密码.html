今天下载了个MySQL的客户端，新建了个数据库，然后开始自己设置密码设得比较私密，在R中连接时（点击r连接MySQL查看如何在R中连接MySQL）密码是显示出来的，所以就更改了下root密码，方法如下，也许以后还要改：

打开cmd命令行，键入mysql -u root，然后使用下面语句进行更改：

SET PASSWORD FOR 'root'@'localhost' = PASSWORD('newpass');

就可以了。 Peace out……
